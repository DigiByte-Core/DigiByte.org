Suunniteltu yhtäjaksoiseen toimintaan.

DigiByte on erittäin turvallinen, monialgoritminen UTXO-lohkoketju, joka on rakennettu skaalattavaksi. 15 sekunnin lohkoajalla ja kahdentoista vuoden katkeamattomalla käyttöajalla verkko tarjoaa kentällä testatun perustan DGB:lle, DigiAssetsille, hajautetulle Digi-ID-tunnistautumiselle ja — aivan pian — DigiDollarille.

Miksi DigiByte

Suunniteltu kryptojen seuraavalle vuosikymmenelle.

Puhdas proof-of-work UTXO-ketju, joka on tuottanut alan ensimmäisiä saavutuksia vuodesta 2014 — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets ja Digi-ID.

15s
Lohkoaika

15 sekunnin lohkot, nopeat vahvistukset.

40× nopeampi kuin Bitcoin ja 10× nopeampi kuin Litecoin. Viidentoista sekunnin vahvistukset tarkoittavat, että maksupäätteen maksut, pörssitalletukset ja varojen siirrot saadaan päätökseen siinä ajassa, kun luet tämän lauseen — ilman hitaiden ketjujen ruuhkia tai palkkiopiikkejä.

5
Konsensus

Viisi louhinta-algoritmia rinnakkain.

DigiByte suojaa lohkot viidellä riippumattomalla proof-of-work-algoritmilla — SHA-256d, Scrypt, Skein, Qubit ja Odocrypt. Jokainen kohdistuu eri louhintaekosysteemiin (CPU, GPU, FPGA, ASIC), ja Odocrypt arpoo itsensä uudelleen 10 päivän välein pitääkseen ASIC-dominoinnin loitolla. Ketjun hyökkääminen tarkoittaa viiden algoritmin samanaikaista hyökkäystä.

21B
Tarjonta

Kova katto, niukkuus suunniteltuna.

DGB on matemaattisesti rajoitettu 21 miljardiin kolikkoon, jotka lasketaan liikkeelle läpinäkyvän, kuukausittain vähenevän aikataulun mukaan. Ei kehittäjäveroa, ei hallintatason avauksia, ei piileviä jakoja. Ennustettava niukkuus tekee rahallisesta omaisuudesta uskottavan — siksi jokainen tavu DGB:tä kantaa todistettavaa, vahvistettavaa arvoa ajan myötä.

12 vuotta käytössä
Saavutukset

12 vuotta käytössä. Nolla käyttökatkoa.

Tammikuusta 2014 mainnet on tuottanut lohkon viidentoista sekunnin välein katkeamatta. Ei syviä uudelleenjärjestelyjä, ei hallinnollisia haaroja, ei hätä-hardforkkeja. Tämä toimintavarmuus yhdistettynä tuhansiin itsenäisesti operoituihin noodeihin ympäri maailmaa on juuri sellainen toiminnallinen näyttö, jota kriittinen infrastruktuuri vaatii — ja jota lähes mikään muu ketju ei pysty esittämään.

DigiAssets

Laske liikkeelle omia varoja turvallisessa UTXO-ketjussa.

Laske liikkeelle natiivisti omia digitaalisia varoja DigiByte-verkossa — tokeneista ja stablecoineista identiteettitietoihin. Jokainen vara selviää 15 sekunnissa ja sitä suojaa maailmanlaajuinen proof-of-work-konsensus.

Valitse varasi

Vaihtokelpoinen token, keräilyesine, sertifikaatti, kuponki — mikä tahansa, mitä voi lyödä UTXO-peruskerrokseen.

Määrittele metatiedot

Aseta nimi, tarjonta, jaettavuus ja IPFS-ankkuroidut metatiedot, jotta lompakot ja selaimet voivat näyttää sen natiivisti.

Allekirjoita & lähetä

Maksa muutama satoshi DGB:tä palkkioksi, lähetä tapahtuma, ja vara on käytössä koko verkossa.

Lähetä minne tahansa

Siirrä mihin tahansa DGB-osoitteeseen. Integroitu vakiona DigiByte Mobileen, DigiAssetX:ään, web-lompakoihin ja selaimiin.

Digi-ID

Salasanaton, avoimen lähdekoodin tunnistautuminen.

Digi-ID perustuu hajautettuihin kryptografisiin allekirjoituksiin perinteisten salasanojen sijaan ja suojaa käyttäjän identiteetin ilman keskitettyjä välittäjiä.

  • Avainten paikallinen säilytys täydelliseen yksityisyyteen.
  • Suunnittelultaan natiivisti phishing-kestävä.
  • Drop-in-integraatio digitaalisille ja fyysisille pääsyille.
Avaa Digi-ID.io Kirjastot
Miten se eroaa OAuthista tai ”Kirjaudu X:llä”?

Perinteinen single sign-on nojaa keskitettyihin datavälittäjiin. Digi-ID on täysin hajautettu: käyttäjät tunnistautuvat allekirjoittamalla yksilöllisen haasteen suoraan laitteellaan, mikä poistaa kolmannen osapuolen palvelimet kokonaan prosessista.

Tarvitseeko kryptolompakon?

Ei, kryptojen omistaminen on vapaaehtoista. Monet DigiByte-lompakot tukevat Digi-ID:tä natiivisti, mutta voit käyttää myös erillistä tunnistautumissovellusta, joka on suunniteltu vain turvallisiin kirjautumisiin.

Miten integroin sen?

Kehittäjät voivat käyttää drop-in-kirjastojamme Node.js:lle, PHP:lle, Pythonille ja Golle, tai asentaa natiivin WordPress-lisäosan. Täydelliset resurssit löytyvät alla olevasta kehittäjähubista.

Kehittäjähub

Tinkimätön infrastruktuuri UTXO-kehitykseen.

Käytä luotettavaa, kentällä testattua RPC:tä kiinteillä ja ennustettavilla palkkiorakenteilla. DigiByte tarjoaa perustan, jolla noodit voi ottaa käyttöön minuuteissa ja rakentaa aidosti hajautetussa verkossa.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

Miksi julkaista DGB:llä?
  • Ennustettavat palkkiomarkkinat: alle sentin transaktiokustannukset mahdollistavat sovellusten kestävän skaalauksen.
  • Hyväksyntävapaat tapahtumat: natiivi UTXO-mekaniikka kiertää laajojen token-valtuuksien tietoturvariskit.
  • Tilaton suoritus: puhdas UTXO-ympäristö mahdollistaa yksinkertaisen, rinnakkaisen ja koostettavan kehityksen.

Saavutukset

Yli vuosikymmen jatkuvaa toteutusta.

Aikaisesta SegWit-aktivoinnista monialgoritmiseen konsensukseen. Jokainen protokollapäivitys on ollut avointa lähdekoodia, mainnetissä testattu ja toteutettu ilman keskitettyä rahoitusta.

  1. 2014

    Genesis

    Reilu julkistus julkisen louhinnan kautta; 0,5 % premine jaettu arvontoihin ja kehitykseen.

  2. 2014

    DigiShield

    Reaaliaikainen vaikeustason säätö vakaiden lohkoaikojen turvaamiseksi.

  3. 2014

    MultiAlgo

    Viisi rinnakkaista louhinta-algoritmia estämään hashraten keskittymistä.

  4. 2017

    SegWit

    Protokollapäivitys, joka optimoi kapasiteetin ja korjaa tapahtumien muunneltavuuden.

  5. 2019

    Odocrypt & Dandelion++

    ASIC-kestävä konsensus ja yksityisyyttä suojaava tapahtumien reititys.

  6. 2020+

    DigiAssets & Digi-ID

    Ekosysteemin laajuinen integraatio ketjun varoille ja hajautetulle tunnistautumiselle.

  7. 2025

    Core v8.26.2

    Modernisoitu noodi-infrastruktuuri rakennettu Bitcoin Core 26 -kehykselle.

  8. 2026

    DigiDollar Tulossa pian

    Täysin ei-säilyttäjäpohjainen stablecoin, joka tuottaa USD-sidottua likviditeettiä aikalukittujen DGB-reservien kautta.

Yhteisö
