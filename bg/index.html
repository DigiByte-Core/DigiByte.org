Създаден за непрекъсната работа.

DigiByte е изключително защитен UTXO блокчейн с множество алгоритми, изграден за мащаб. С 15-секундно време на блок и дванадесетгодишна история на непрекъсната работа мрежата осигурява изпитана на бойното поле основа за DGB, DigiAssets, децентрализираното удостоверяване Digi-ID и — много скоро — DigiDollar.

Налична в цялата екосистема

Защо DigiByte

Създаден за следващото десетилетие в крипто.

Чиста UTXO верига с proof-of-work, която от 2014 г. доставя индустриални първенства — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets и Digi-ID.

15s
Време на блок

15-секундни блокове, бързо уреждани.

40× по-бърз от Bitcoin и 10× по-бърз от Litecoin. Петнадесетсекундните потвърждения означават, че POS плащанията, депозитите по борси и трансферите на активи се финализират за времето, в което прочитате това изречение — без задръстванията и скоковете на такси на по-бавните вериги.

5
Консенсус

Пет алгоритъма за добив паралелно.

DigiByte осигурява блоковете чрез пет независими proof-of-work алгоритъма — SHA-256d, Scrypt, Skein, Qubit и Odocrypt. Всеки таргетира различна добивна екосистема (CPU, GPU, FPGA, ASIC), а Odocrypt се ре-рандомизира на всеки 10 дни, за да предотврати ASIC-доминация. Атака срещу веригата означава едновременна атака срещу пет алгоритъма.

21B
Предлагане

Твърд таван, оскъдност по дизайн.

DGB е математически ограничен до 21 милиарда монети, емитирани по прозрачен график с месечно намаление. Без такса за разработчиците, без отключвания на управление, без скрити разпределения. Предвидимата оскъдност прави един монетарен актив достоверен — затова всеки байт DGB носи доказуема, проверима стойност във времето.

12 години активна
История

Дванадесет години активна. Нула downtime.

От януари 2014 г. mainnet произвежда блок на всеки петнадесет секунди без прекъсване. Без дълбоки re-orgs, без governance форкове, без аварийни hard форкове. Тази наличност, заедно с хиляди независимо управлявани възли по света, е именно видът оперативна история, която изисква критичната инфраструктура — и която почти никоя друга верига не може да покаже.

Digi-ID

Удостоверяване без парола с отворен код.

Digi-ID разчита на децентрализирани криптографски подписи вместо традиционни пароли, защитавайки идентичността на потребителя без централни посредници.

  • Локално съхранение на ключове за абсолютна поверителност.
  • Нативно устойчив на фишинг дизайн.
  • Drop-in интеграция за цифрови и физически достъпи.
Отвори Digi-ID.io Библиотеки
По какво се различава от OAuth или „Вход с X“?

Традиционното единно влизане разчита на централизирани посредници за данни. Digi-ID е напълно децентрализирано — потребителят се удостоверява с подписване на уникално предизвикателство директно на устройството си, което напълно премахва третите страни от процеса.

Нуждае ли се от крипто портфейл?

Не, притежаването на крипто е по избор. Много DigiByte портфейли поддържат Digi-ID нативно, но можете да използвате и самостоятелно приложение за удостоверяване, предназначено само за сигурни входове.

Как да го интегрирам?

Разработчиците могат да използват нашите drop-in библиотеки за Node.js, PHP, Python и Go, или да инсталират нативния WordPress плъгин. Пълни ресурси ще намерите в хъба за разработчици по-долу.

Хъб за разработчици

Безкомпромисна инфраструктура за UTXO разработка.

Имате достъп до надежден, изпитан RPC с фиксирани, предсказуеми структури на таксите. DigiByte осигурява основата за разгръщане на възли за минути и изграждане върху наистина децентрализирана мрежа.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

История

Над десетилетие непрекъснато изпълнение.

От ранното активиране на SegWit до много-алгоритмения консенсус. Всяка надстройка на протокола е с отворен код, тествана в mainnet и реализирана без централизирано финансиране.

  1. 2014

    Genesis

    Честен старт чрез публично копаене; 0,5 % премайн, разпределен за giveaways и разработка.

  2. 2014

    DigiShield

    Регулиране на трудността в реално време за стабилно време на блока.

  3. 2014

    MultiAlgo

    Пет паралелни алгоритъма за добив, които предотвратяват централизация на хешрейта.

  4. 2017

    SegWit

    Надстройка на протокола, оптимизираща капацитета и фиксираща malleability на транзакциите.

  5. 2019

    Odocrypt & Dandelion++

    ASIC-устойчив консенсус и трасиране на транзакции, пазещо поверителността.

  6. 2020+

    DigiAssets & Digi-ID

    Интеграция в цялата екосистема за активи на веригата и децентрализирано удостоверяване.

  7. 2025

    Core v8.26.2

    Модернизирана инфраструктура на възел, изградена върху рамката Bitcoin Core 26.

  8. 2026

    DigiDollar Скоро

    Изцяло не-кустодиален стейбълкойн, който генерира ликвидност, обвързана с USD, чрез заключени във времето DGB резерви.

Общност

Независима мрежа от глобални строители.

DigiByte е чисто отворен проект без централизирана организация. Глобалната разработка, инфраструктурата на възлите и локализацията на над 40 езика се поддържат изцяло от самокоординиращи се доброволци.
