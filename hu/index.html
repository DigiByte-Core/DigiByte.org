Megszakítás nélküli működésre tervezve.

A DigiByte egy rendkívül biztonságos, többalgoritmusos UTXO blockchain, amelyet skálázásra terveztek. 15 másodperces blokkideje és tizenkét év megszakítás nélküli rendelkezésre állása révén a hálózat csatában bizonyított alapot ad a DGB-nek, a DigiAssets-nek, a decentralizált Digi-ID hitelesítésnek és — hamarosan — a DigiDollarnak.

Miért DigiByte

A kriptó következő évtizedére tervezve.

Tiszta proof-of-work UTXO lánc, amely 2014 óta iparági úttöréseket szállít — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets és Digi-ID.

15s
Blokkidő

15 másodperces blokkok, gyors elszámolás.

40× gyorsabb a Bitcoinnál és 10× gyorsabb a Litecoinnál. A tizenöt másodperces megerősítések azt jelentik, hogy a POS fizetések, tőzsdei befizetések és eszközátutalások annyi idő alatt zárulnak le, amennyi ennek a mondatnak az elolvasásához kell — a lassabb láncokra jellemző torlódások és díjnövekedés nélkül.

5
Konszenzus

Öt bányász-algoritmus párhuzamosan.

A DigiByte öt független proof-of-work algoritmussal — SHA-256d, Scrypt, Skein, Qubit és Odocrypt — védi a blokkokat. Mindegyik egy másik bányász-ökoszisztémára (CPU, GPU, FPGA, ASIC) irányul, és az Odocrypt 10 naponta véletlenszerűsíti újra magát, hogy elkerülje az ASIC dominanciát. A lánc támadása öt algoritmus egyidejű támadását jelenti.

21B
Kínálat

Kemény plafon, tervezett ritkaság.

A DGB matematikailag 21 milliárd érmére van korlátozva, amelyek átlátható havi csökkentésű menetrend szerint kerülnek kibocsátásra. Nincs fejlesztői adó, nincsenek kormányzási feloldások, nincsenek rejtett kiosztások. A kiszámítható ritkaság teszi hitelessé a monetáris eszközt — ezért minden bájt DGB bizonyítható, ellenőrizhető értéket hordoz az idő múlásával.

12 év üzemelés
Múlt

Tizenkét év üzemelés. Nulla leállás.

2014 januárja óta a mainnet minden tizenöt másodpercben blokkokat termel megszakítás nélkül. Nincsenek mély re-orgok, nincsenek kormányzási forkok, nincsenek vészhelyzeti hard forkok. Ez az üzemidő, a világszerte függetlenül üzemeltetett több ezer csomóponttal együtt, pontosan az a működési múlt, amelyre a kritikus infrastruktúrának szüksége van — és amelyet szinte semmilyen más lánc nem tud felmutatni.

Digi-ID

Jelszó nélküli, nyílt forráskódú hitelesítés.

A Digi-ID a hagyományos jelszavak helyett decentralizált kriptográfiai aláírásokra épül, megvédve a felhasználó identitását központi közvetítők nélkül.

  • Helyi kulcs-tárolás abszolút adatvédelemért.
  • Natívan adathalászatnak ellenálló kialakítás.
  • Drop-in integráció digitális és fizikai hozzáférésekhez.
Digi-ID.io megnyitása Könyvtárak
Miben különbözik az OAuth-tól vagy a „Bejelentkezés X-szel” módoktól?

A hagyományos egyszeri bejelentkezés központi adatközvetítőkre épül. A Digi-ID teljesen decentralizált: a felhasználók egy egyedi kihíváshoz közvetlenül a saját eszközükön írnak alá, így a folyamatból teljesen kikerülnek a harmadik feles szerverek.

Szükség van kriptotárcára?

Nem, kriptó birtoklása opcionális. Sok DigiByte tárca natívan támogatja a Digi-ID-t, de használhatsz egy önálló hitelesítő alkalmazást is, amely kizárólag biztonságos bejelentkezésekre készült.

Hogyan integrálom?

A fejlesztők használhatják a Node.js-hez, PHP-hez, Pythonhoz és Go-hoz készült drop-in könyvtárainkat, vagy telepíthetik a natív WordPress bővítményt. A teljes erőforrások az alábbi fejlesztői hubban találhatók.

Fejlesztői hub

Megalkuvás nélküli infrastruktúra UTXO fejlesztéshez.

Hozzáférés megbízható, csatában tesztelt RPC-hez, fix és kiszámítható díjszerkezettel. A DigiByte alapot ad ahhoz, hogy perceken belül csomópontokat telepíts és valóban decentralizált hálózaton építkezz.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

Múlt

Egy évtizednél is több folyamatos megvalósítás.

A SegWit korai bevezetésétől a többalgoritmusos konszenzusig. Minden protokollfrissítés nyílt forráskódú, mainnetben tesztelt és központi finanszírozás nélkül telepített.

  1. 2014

    Genesis

    Fair indulás nyilvános bányászattal; 0,5 % premine giveaway-ekre és fejlesztésre osztva.

  2. 2014

    DigiShield

    Valós idejű nehézségbeállítás a stabil blokkidőkért.

  3. 2014

    MultiAlgo

    Öt párhuzamos bányász-algoritmus a hashrate központosulásának megakadályozására.

  4. 2017

    SegWit

    Protokollfrissítés, amely optimalizálja a kapacitást és kijavítja a tranzakciós malleability-t.

  5. 2019

    Odocrypt & Dandelion++

    ASIC-ellenálló konszenzus és adatvédelmet megőrző tranzakciós útválasztás.

  6. 2020+

    DigiAssets & Digi-ID

    Ökoszisztéma-szintű integráció on-chain eszközökhöz és decentralizált hitelesítéshez.

  7. 2025

    Core v9.26.4

    Modernizált csomópont-infrastruktúra a Bitcoin Core 26 keretrendszerre építve.

    • BIP340 — Schnorr signatures
    • BIP341 — Pay-to-Taproot (P2TR)
    • BIP342 — Tapscript
  8. 2026

    DigiDollar Live on mainnet

    Teljesen nem-letétkezelő stablecoin, amely USD-hez kötött likviditást generál időzárolt DGB tartalékokon keresztül.

Globális építők független hálózata.

A DigiByte tiszta nyílt forráskódú projekt, központosított szervezet nélkül. A globális fejlesztést, csomópont-infrastruktúrát és több mint 40 nyelvre szóló honosítást teljes egészében önszervező önkéntesek tartják fenn.
