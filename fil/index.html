Digi-ID

Sign in with a scan. No password, no broker.

Digi-ID is open-source, phishing-resistant, decentralized authentication. Your keys, your identity — your users never re-use a password again.

  • Private — keys live only on the user's device.
  • Phishing-resistant by cryptographic design.
  • Drop-in for web, mobile, IoT and physical access.
Open Digi-ID.io Libraries
How is it different from OAuth or "Sign in with X"?

Digi-ID has no central identity provider. There is no Google, Apple, or Meta in the loop — the user signs a one-time challenge with their own key, end of story.

Does it require a wallet?

It only requires a Digi-ID-compatible app. Many DigiByte wallets include support, and standalone authenticators are available too.

How do I integrate it?

Drop-in libraries for Node.js, PHP, Python, Go and a WordPress plugin are linked in the developer hub below.

Build on the longest UTXO chain in production.

Battle-tested RPC, predictable fees, no token approvals, no rug-pulls. Get a node running in five minutes and ship something real.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)
