Konstruerad för oavbruten drift.

DigiByte är en mycket säker UTXO-blockkedja med flera algoritmer, byggd för skalning. Med 15 sekunders blocktid och tolv år av oavbruten drift utgör nätverket en stridserfaren grund för DGB, DigiAssets, decentraliserad Digi-ID-autentisering och — väldigt snart — DigiDollar.

Varför DigiByte

Konstruerad för kryptons nästa decennium.

En ren proof-of-work UTXO-kedja som levererat branschpremiärer sedan 2014 — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets och Digi-ID.

15s
Blocktid

15-sekunders block, snabbt slutförda.

40× snabbare än Bitcoin och 10× snabbare än Litecoin. Femton sekunders bekräftelser betyder att kassabetalningar, insättningar på börser och tillgångsöverföringar slutförs på den tid det tar att läsa denna mening — utan långsamma kedjors trängsel eller avgiftsspikar.

5
Konsensus

Fem mining-algoritmer parallellt.

DigiByte säkrar block med fem oberoende proof-of-work-algoritmer — SHA-256d, Scrypt, Skein, Qubit och Odocrypt. Var och en riktar sig till olika mining-ekosystem (CPU, GPU, FPGA, ASIC) och Odocrypt slumpar om sig själv var tionde dag för att hålla ASIC-dominans utanför. Att attackera kedjan innebär att attackera fem algoritmer samtidigt.

21B
Utbud

Hårt begränsat, knappt by design.

DGB är matematiskt begränsat till 21 miljarder mynt, utgivna enligt ett transparent schema med månatlig minskning. Ingen utvecklarskatt, inga governance-upplåsningar, inga dolda allokeringar. Förutsägbar knapphet gör en monetär tillgång trovärdig — och därför bär varje byte DGB påvisbart, verifierbart värde över tid.

12 år live
Track record

Tolv år live. Noll driftstopp.

Sedan januari 2014 har mainnet producerat block var femtonde sekund utan avbrott. Inga djupa re-orgs, inga governance-forkar, inga nöd-hard forks. Den drifttiden, tillsammans med tusentals oberoende drivna noder världen över, är precis den typ av driftshistorik som kritisk infrastruktur kräver — och som nästan ingen annan kedja kan visa upp.

DigiAssets

Utfärda egna tillgångar på en säker UTXO-kedja.

Utfärda egna digitala tillgångar nativt på DigiByte-nätverket — från tokens och stablecoins till identitetsposter. Varje tillgång slutförs på 15 sekunder och säkras av en global proof-of-work-konsensus.

Välj din tillgång

Utbytbar token, samlarobjekt, certifikat, voucher — allt som kan myntas på ett UTXO-baslager.

Definiera metadata

Ange namn, utbud, delbarhet och IPFS-förankrad metadata så att plånböcker och utforskare kan visa det nativt.

Signera & sänd

Betala ett par sats DGB i avgift, sänd transaktionen och tillgången är aktiv i hela nätverket.

Digi-ID

Lösenordsfri, open source-autentisering.

Digi-ID bygger på decentraliserade kryptografiska signaturer i stället för traditionella lösenord, och säkrar användarens identitet utan centrala mellanhänder.

  • Lokal nyckellagring för absolut integritet.
  • Inbyggt phishing-resistent design.
  • Drop-in-integration för digital och fysisk åtkomst.
Öppna Digi-ID.io Bibliotek
Hur skiljer det sig från OAuth eller ”Logga in med X”?

Traditionell single sign-on förlitar sig på centrala datamellanhänder. Digi-ID är helt decentraliserat: användare autentiserar genom att signera en unik utmaning direkt från sin enhet, vilket helt tar bort tredjepartsservrar från processen.

Krävs en kryptoplånbok?

Nej, att äga krypto är valfritt. Många DigiByte-plånböcker stöder Digi-ID nativt, men du kan också använda en fristående autentiseringsapp som är gjord enbart för säkra inloggningar.

Hur integrerar jag det?

Utvecklare kan använda våra drop-in-bibliotek för Node.js, PHP, Python och Go, eller installera den inbyggda WordPress-pluginen. Fullständiga resurser finns i utvecklarhubben nedan.

Utvecklarhubb

Kompromisslös infrastruktur för UTXO-utveckling.

Få tillgång till en pålitlig, stridserfaren RPC med fasta, förutsägbara avgiftsstrukturer. DigiByte ger grunden för att rulla ut noder på minuter och bygga på ett verkligt decentraliserat nätverk.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

Varför leverera på DGB?
  • Förutsägbara avgiftsmarknader: transaktionskostnader under en cent möjliggör hållbar app-skalning.
  • Approval-fria transaktioner: den inbyggda UTXO-mekaniken undviker säkerhetsriskerna med breda token-godkännanden.
  • Stateless exekvering: en ren UTXO-miljö gör enkel, parallell och komponerbar utveckling möjlig.
Track record

Över ett decennium av kontinuerlig leverans.

Från tidig SegWit-aktivering till multi-algoritm-konsensus. Varje protokolluppgradering har varit open source, mainnet-testad och utrullad utan central finansiering.

  1. 2014

    Genesis

    Rättvis start via offentlig mining; 0,5 % premine fördelad för giveaways och utveckling.

  2. 2014

    DigiShield

    Realtidsjustering av svårighetsgraden för stabila blocktider.

  3. 2014

    MultiAlgo

    Fem parallella mining-algoritmer för att förhindra hashrate-centralisering.

  4. 2017

    SegWit

    Protokolluppgradering som optimerar kapacitet och åtgärdar transaktionsmalleabilitet.

  5. 2019

    Odocrypt & Dandelion++

    ASIC-resistent konsensus och integritetsbevarande transaktionsrouting.

  6. 2020+

    DigiAssets & Digi-ID

    Ekosystemtäckande integration för on-chain-tillgångar och decentraliserad autentisering.

  7. 2025

    Core v9.26.4

    Moderniserad nod-infrastruktur byggd på Bitcoin Core 26-ramverket.

  8. 2026

    DigiDollar Snart tillgänglig

    En helt non-custodial stablecoin som genererar USD-bunden likviditet via tidslåsta DGB-reserver.

Gemenskap

Ett oberoende nätverk av globala byggare.

DigiByte är ett rent open source-projekt utan central organisation. Global utveckling, nodinfrastruktur och lokalisering till över 40 språk underhålls helt av självsamordnande volontärer.

Bli involverad: bidra till ekosystemet genom att granska kod, lokalisera dokumentation, driva infrastruktur eller skapa utbildningsinnehåll.
