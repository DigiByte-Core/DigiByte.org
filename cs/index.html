Navržen pro nepřetržitý provoz.

DigiByte je vysoce zabezpečený UTXO blockchain s více algoritmy, postavený pro škálování. S patnáctisekundovým časem bloku a dvanáctiletou historií nepřerušeného provozu poskytuje síť bojem prověřený základ pro DGB, DigiAssets, decentralizované ověřování Digi-ID a — velmi brzy — DigiDollar.

Proč DigiByte

Navržen pro další dekádu krypta.

Čistý proof-of-work UTXO řetězec, který od roku 2014 přináší prvenství v odvětví — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets a Digi-ID.

15s
Čas bloku

15sekundové bloky, rychlé vyrovnání.

40× rychlejší než Bitcoin a 10× rychlejší než Litecoin. Patnáctisekundová potvrzení znamenají, že POS platby, vklady na burzy a převody aktiv se finalizují za čas, který vám zabere přečtení této věty — bez přetížení nebo nárůstu poplatků pomalejších řetězců.

5
Konsenzus

Pět těžebních algoritmů paralelně.

DigiByte zabezpečuje bloky pěti nezávislými proof-of-work algoritmy — SHA-256d, Scrypt, Skein, Qubit a Odocrypt. Každý cílí na jiný těžební ekosystém (CPU, GPU, FPGA, ASIC) a Odocrypt se každých 10 dní sám re-randomizuje, aby vyloučil dominanci ASIC. Útok na řetězec znamená útok na pět algoritmů najednou.

21B
Nabídka

Pevný strop, vzácnost záměrná.

DGB je matematicky omezeno na 21 miliard mincí emitovaných podle transparentního měsíčně se snižujícího plánu. Žádná dev daň, žádná governance odemčení, žádné skryté alokace. Předvídatelná vzácnost dělá monetární aktivum důvěryhodným — a proto každý bajt DGB nese prokazatelnou, ověřitelnou hodnotu v čase.

12 let v provozu
Historie

Dvanáct let v provozu. Nulový výpadek.

Od ledna 2014 mainnet produkuje bloky každých patnáct sekund bez přerušení. Žádné hluboké re-orgs, žádné governance forky, žádné nouzové hard forky. Tato dostupnost spolu s tisíci nezávisle provozovaných uzlů po celém světě je přesně ten typ provozního záznamu, který kritická infrastruktura potřebuje — a kterým se téměř žádný jiný řetězec nemůže pochlubit.

DigiAssets

Vydejte vlastní aktiva na bezpečném UTXO řetězci.

Vydávejte nativně vlastní digitální aktiva na síti DigiByte — od tokenů a stablecoinů po záznamy identity. Každé aktivum se vyřídí za 15 sekund a je chráněno globálním proof-of-work konsenzem.

Digi-ID

Ověřování bez hesla, open source.

Digi-ID staví na decentralizovaných kryptografických podpisech místo tradičních hesel a chrání identitu uživatele bez centralizovaných prostředníků.

  • Lokální úložiště klíčů pro absolutní soukromí.
  • Návrh nativně odolný proti phishingu.
  • Drop-in integrace pro digitální i fyzické přístupy.
Otevřít Digi-ID.io Knihovny
Čím se liší od OAuth nebo „Přihlásit se přes X“?

Klasické single sign-on stojí na centralizovaných datových prostřednících. Digi-ID je plně decentralizované: uživatel se ověřuje podepsáním unikátní výzvy přímo na svém zařízení, čímž jsou servery třetích stran z procesu zcela odstraněny.

Vyžaduje to kryptopeněženku?

Ne, vlastnictví kryptoměn je volitelné. Mnoho DigiByte peněženek podporuje Digi-ID nativně, ale lze využít i samostatnou autentifikační aplikaci určenou výhradně pro bezpečná přihlášení.

Jak to zaintegrovat?

Vývojáři mohou použít naše drop-in knihovny pro Node.js, PHP, Python a Go, nebo nainstalovat nativní WordPress plugin. Kompletní zdroje najdete v hubu pro vývojáře níže.

Hub pro vývojáře

Bezkompromisní infrastruktura pro UTXO vývoj.

Přístup ke spolehlivému, bojem prověřenému RPC s pevnými, předvídatelnými poplatky. DigiByte poskytuje základ pro nasazení uzlů během minut a vývoj nad opravdu decentralizovanou sítí.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)
curl -u rpcuser:rpcpass --data-binary \

Proč nasazovat na DGB?
  • Předvídatelné trhy s poplatky: transakční náklady pod jeden cent umožňují udržitelné škálování aplikací.
  • Transakce bez approvals: nativní mechanika UTXO obchází bezpečnostní rizika spojená s rozsáhlými povoleními tokenů.
  • Bezstavové provádění: čisté UTXO prostředí umožňuje jednoduchý, paralelní a komponovatelný vývoj.
Historie

Přes deset let nepřetržitého doručování.

Od raného nasazení SegWit po vícealgoritmický konsenzus. Každá upgrade protokolu byl open source, otestován na mainnetu a nasazen bez centralizovaného financování.

  1. 2014

    Genesis

    Férový start prostřednictvím veřejné těžby; 0,5 % premine rozdělené na giveawaye a vývoj.

  2. 2014

    DigiShield

    Úprava obtížnosti v reálném čase pro stabilní časy bloků.

  3. 2014

    MultiAlgo

    Pět paralelních těžebních algoritmů zamezujících centralizaci hashrate.

  4. 2017

    SegWit

    Upgrade protokolu optimalizující kapacitu a opravující malleability transakcí.

  5. 2019

    Odocrypt & Dandelion++

    Konsenzus odolný proti ASIC a směrování transakcí zachovávající soukromí.

  6. 2020+

    DigiAssets & Digi-ID

    Celosystémová integrace pro on-chain aktiva a decentralizované ověřování.

  7. 2025

    Core v9.26.4

    Modernizovaná infrastruktura uzlu postavená na frameworku Bitcoin Core 26.

    • BIP340 — Schnorr signatures
    • BIP341 — Pay-to-Taproot (P2TR)
    • BIP342 — Tapscript
  8. 2026

    DigiDollar Live on mainnet

    Plně non-custodial stablecoin generující likviditu navázanou na USD prostřednictvím časově zamčených DGB rezerv.

Komunita

Nezávislá síť globálních tvůrců.

DigiByte je čistě open-source projekt bez centralizované organizace. Globální vývoj, infrastrukturu uzlů a lokalizaci do více než 40 jazyků udržují výhradně samokoordinovaní dobrovolníci.

Zapojte se: připojte se k ekosystému auditováním kódu, lokalizací dokumentace, provozováním infrastruktury nebo tvorbou vzdělávacího obsahu.

Per DIP-0002, third-party social platforms and advocacy groups have been relocated to the community-maintained DigiByte Wiki and the official GitHub Discussions forum.
