Σχεδιασμένο για συνεχή λειτουργία.

Το DigiByte είναι ένα εξαιρετικά ασφαλές, πολυαλγοριθμικό UTXO blockchain χτισμένο για κλίμακα. Με μπλοκ κάθε 15 δευτερόλεπτα και δώδεκα χρόνια αδιάκοπης λειτουργίας, το δίκτυο προσφέρει δοκιμασμένο στην πράξη θεμέλιο για το DGB, τα DigiAssets, την αποκεντρωμένη αυθεντικοποίηση Digi-ID και — πολύ σύντομα — το DigiDollar.

Γιατί DigiByte

Σχεδιασμένο για την επόμενη δεκαετία του κρύπτο.

Μια καθαρή proof-of-work UTXO αλυσίδα που από το 2014 φέρνει πρωτιές στον κλάδο — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets και Digi-ID.

15s
Χρόνος μπλοκ

Μπλοκ 15 δευτερολέπτων, ταχεία διευθέτηση.

40× ταχύτερο από το Bitcoin και 10× ταχύτερο από το Litecoin. Οι επιβεβαιώσεις δεκαπέντε δευτερολέπτων σημαίνουν ότι πληρωμές στο ταμείο, καταθέσεις σε ανταλλακτήρια και μεταφορές περιουσιακών στοιχείων ολοκληρώνονται μέσα στον χρόνο που χρειάζεται για να διαβάσετε αυτή την πρόταση — χωρίς τη συμφόρηση ή τα άλματα χρεώσεων των πιο αργών αλυσίδων.

5
Συναίνεση

Πέντε αλγόριθμοι εξόρυξης παράλληλα.

Το DigiByte ασφαλίζει τα μπλοκ μέσω πέντε ανεξάρτητων αλγορίθμων proof-of-work — SHA-256d, Scrypt, Skein, Qubit και Odocrypt. Καθένας στοχεύει σε διαφορετικό οικοσύστημα εξόρυξης (CPU, GPU, FPGA, ASIC) και ο Odocrypt αυτο-τυχαιοποιείται κάθε 10 ημέρες ώστε να αποκλείει την κυριαρχία των ASIC. Το να επιτεθεί κανείς στην αλυσίδα σημαίνει να επιτεθεί ταυτόχρονα σε πέντε αλγορίθμους.

21B
Προσφορά

Σκληρό όριο, σπανιότητα εκ σχεδιασμού.

Το DGB είναι μαθηματικά περιορισμένο σε 21 δισ. νομίσματα, εκδίδονται με διαφανές πρόγραμμα μηνιαίας μείωσης. Χωρίς φόρο για προγραμματιστές, χωρίς ξεκλειδώματα διακυβέρνησης, χωρίς κρυφές κατανομές. Η προβλέψιμη σπανιότητα καθιστά ένα νομισματικό στοιχείο αξιόπιστο — γι' αυτό κάθε byte DGB φέρει αποδείξιμη, επαληθεύσιμη αξία στον χρόνο.

12 έτη ενεργό
Ιστορικό

Δώδεκα χρόνια ενεργό. Μηδέν χρόνος εκτός.

Από τον Ιανουάριο του 2014 το mainnet παράγει μπλοκ κάθε δεκαπέντε δευτερόλεπτα χωρίς διακοπή. Χωρίς βαθιά re-orgs, χωρίς forks διακυβέρνησης, χωρίς επείγοντα hard forks. Αυτή η διαθεσιμότητα, μαζί με χιλιάδες ανεξάρτητα λειτουργούντα κόμβους παγκοσμίως, είναι ακριβώς το είδος λειτουργικού ιστορικού που χρειάζεται η κρίσιμη υποδομή — και που σχεδόν καμία άλλη αλυσίδα δεν μπορεί να επιδείξει.

DigiAssets

Έκδοση προσαρμοσμένων στοιχείων σε μια ασφαλή UTXO αλυσίδα.

Εκδώστε εγγενώς προσαρμοσμένα ψηφιακά στοιχεία στο δίκτυο DigiByte — από tokens και stablecoins έως εγγραφές ταυτότητας. Κάθε στοιχείο διευθετείται σε 15 δευτερόλεπτα και ασφαλίζεται από παγκόσμια συναίνεση proof-of-work.

Επιλέξτε το στοιχείο σας

Ανταλλάξιμο token, συλλεκτικό, πιστοποιητικό, voucher — οτιδήποτε μπορεί να κοπεί σε ένα UTXO βασικό επίπεδο.

Digi-ID

Αυθεντικοποίηση ανοιχτού κώδικα, χωρίς κωδικό πρόσβασης.

Το Digi-ID βασίζεται σε αποκεντρωμένες κρυπτογραφικές υπογραφές αντί για παραδοσιακούς κωδικούς, προστατεύοντας την ταυτότητα του χρήστη χωρίς κεντρικούς μεσάζοντες.

  • Τοπική αποθήκευση κλειδιών για απόλυτη ιδιωτικότητα.
  • Σχεδιασμός εγγενώς ανθεκτικός σε phishing.
  • Drop-in ενσωμάτωση για ψηφιακές και φυσικές προσβάσεις.
Άνοιξε το Digi-ID.io Βιβλιοθήκες
Σε τι διαφέρει από το OAuth ή το «Σύνδεση με X»;

Η παραδοσιακή single sign-on βασίζεται σε κεντρικούς μεσάζοντες δεδομένων. Το Digi-ID είναι πλήρως αποκεντρωμένο: ο χρήστης αυθεντικοποιείται υπογράφοντας μια μοναδική πρόκληση απευθείας στη συσκευή του, αφαιρώντας τους τρίτους servers εντελώς από τη διαδικασία.

Απαιτείται πορτοφόλι κρυπτογραφημάτων;

Όχι, η κατοχή κρύπτο είναι προαιρετική. Πολλά πορτοφόλια DigiByte υποστηρίζουν εγγενώς το Digi-ID, αλλά μπορείτε να χρησιμοποιήσετε και μια αυτόνομη εφαρμογή αυθεντικοποίησης σχεδιασμένη αποκλειστικά για ασφαλείς συνδέσεις.

Πώς το ενσωματώνω;

Οι προγραμματιστές μπορούν να χρησιμοποιήσουν τις drop-in βιβλιοθήκες μας για Node.js, PHP, Python και Go, ή να εγκαταστήσουν το εγγενές WordPress plugin. Πλήρεις πόροι βρίσκονται στο developer hub παρακάτω.

Άκαμπτη υποδομή για ανάπτυξη UTXO.

Αποκτήστε πρόσβαση σε αξιόπιστο, δοκιμασμένο στη μάχη RPC με σταθερές, προβλέψιμες δομές χρεώσεων. Το DigiByte παρέχει τη βάση για να αναπτύξετε κόμβους σε λίγα λεπτά και να χτίσετε πάνω σε ένα πραγματικά αποκεντρωμένο δίκτυο.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)
curl -u rpcuser:rpcpass --data-binary \

Ιστορικό

Πάνω από μια δεκαετία συνεχούς εκτέλεσης.

Από την πρώιμη ενεργοποίηση του SegWit έως τη συναίνεση πολλαπλών αλγορίθμων. Κάθε αναβάθμιση πρωτοκόλλου ήταν ανοιχτού κώδικα, δοκιμασμένη στο mainnet και παραδομένη χωρίς κεντρική χρηματοδότηση.

  1. 2014

    Genesis

    Δίκαιη εκκίνηση μέσω δημόσιας εξόρυξης· 0,5 % premine μοιρασμένο σε giveaways και ανάπτυξη.

  2. 2014

    DigiShield

    Πραγματικού χρόνου ρύθμιση δυσκολίας για σταθερούς χρόνους μπλοκ.

  3. 2014

    MultiAlgo

    Πέντε παράλληλοι αλγόριθμοι εξόρυξης για την αποφυγή συγκέντρωσης hashrate.

  4. 2017

    SegWit

    Αναβάθμιση πρωτοκόλλου που βελτιστοποιεί τη χωρητικότητα και διορθώνει την malleability συναλλαγών.

  5. 2019

    Odocrypt & Dandelion++

    Ανθεκτική σε ASIC συναίνεση και δρομολόγηση συναλλαγών που διατηρεί την ιδιωτικότητα.

  6. 2020+

    DigiAssets & Digi-ID

    Ενσωμάτωση σε επίπεδο οικοσυστήματος για on-chain στοιχεία και αποκεντρωμένη αυθεντικοποίηση.

  7. 2025

    Core v9.26.4

    Εκσυγχρονισμένη υποδομή κόμβου χτισμένη στο πλαίσιο Bitcoin Core 26.

    • BIP340 — Schnorr signatures
    • BIP341 — Pay-to-Taproot (P2TR)
    • BIP342 — Tapscript
  8. 2026

    DigiDollar Live on mainnet

    Ένα πλήρως μη επιτηρούμενο stablecoin που δημιουργεί ρευστότητα συνδεδεμένη με USD μέσω χρονικά κλειδωμένων αποθεμάτων DGB.

Κοινότητα

Ένα ανεξάρτητο δίκτυο παγκόσμιων δημιουργών.

Το DigiByte είναι ένα καθαρά έργο ανοιχτού κώδικα χωρίς κεντρική οργάνωση. Η παγκόσμια ανάπτυξη, η υποδομή κόμβων και η μετάφραση σε πάνω από 40 γλώσσες συντηρούνται εξ ολοκλήρου από αυτο-συντονισμένους εθελοντές.

Συμμετοχή: γίνετε μέρος του οικοσυστήματος ελέγχοντας κώδικα, μεταφράζοντας τεκμηρίωση, διαχειριζόμενοι υποδομή ή παράγοντας εκπαιδευτικό υλικό.

Per DIP-0002, third-party social platforms and advocacy groups have been relocated to the community-maintained DigiByte Wiki and the official GitHub Discussions forum.
