Construit pentru funcționare neîntreruptă.

DigiByte este o blockchain UTXO multi-algoritm, foarte sigură, proiectată pentru scalare. Cu blocuri la 15 secunde și doisprezece ani de funcționare neîntreruptă, rețeaua oferă o bază testată în luptă pentru DGB, DigiAssets, autentificarea descentralizată Digi-ID și — foarte curând — DigiDollar.

De ce DigiByte

Construită pentru următorul deceniu al cripto.

Un lanț UTXO pur proof-of-work care livrează premiere de industrie din 2014 — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets și Digi-ID.

15s
Timp de bloc

Blocuri de 15 s, decontate rapid.

De 40× mai rapid decât Bitcoin și de 10× mai rapid decât Litecoin. Confirmările în cincisprezece secunde înseamnă că plățile POS, depunerile pe exchange-uri și transferurile de active se finalizează cât citești această propoziție — fără congestia sau saltul comisioanelor de pe lanțurile mai lente.

5
Consens

Cinci algoritmi de minat în paralel.

DigiByte protejează blocurile prin cinci algoritmi proof-of-work independenți — SHA-256d, Scrypt, Skein, Qubit și Odocrypt. Fiecare vizează un ecosistem diferit de minat (CPU, GPU, FPGA, ASIC), iar Odocrypt se re-randomizează la fiecare 10 zile pentru a împiedica dominația ASIC-urilor. A ataca lanțul înseamnă a ataca cinci algoritmi simultan.

21B
Ofertă

Plafon fix, rar prin design.

DGB este limitat matematic la 21 de miliarde de monede, emise pe un calendar transparent cu reducere lunară. Fără taxă pentru dezvoltatori, fără deblocări de guvernanță, fără alocări ascunse. Raritatea previzibilă face un activ monetar credibil — și de aceea fiecare octet de DGB poartă valoare demonstrabilă, verificabilă în timp.

12 ani activă
Istoric

Doisprezece ani activă. Zero perioade de pauză.

Din ianuarie 2014, mainnet-ul produce blocuri la fiecare cincisprezece secunde, fără întrerupere. Fără reorgs profunde, fără fork-uri de guvernanță, fără hard fork-uri de urgență. Acest uptime, alături de mii de noduri operate independent în lume, este genul de istoric operațional de care are nevoie infrastructura critică — și pe care aproape niciun alt lanț nu îl poate arăta.

DigiAssets

Emite active personalizate pe un lanț UTXO sigur.

Emite native active digitale personalizate pe rețeaua DigiByte, de la token-uri și stablecoin-uri până la înregistrări de identitate. Fiecare activ este decontat în 15 secunde și securizat de un consens global proof-of-work.

Alege activul

Token fungibil, obiect de colecție, certificat, voucher — orice se poate emite pe un strat de bază UTXO.

Definește metadatele

Setează nume, ofertă, divizibilitate și metadate ancorate în IPFS pentru ca portofelele și explorerele să le afișeze nativ.

Semnează & difuzează

Plătește câțiva sați de DGB drept taxe, difuzează tranzacția și activul este viu în toată rețeaua.

Digi-ID

Autentificare open source, fără parolă.

Digi-ID se bazează pe semnături criptografice descentralizate în locul parolelor tradiționale, protejând identitatea utilizatorului fără intermediari centralizați.

  • Stocare locală a cheilor pentru confidențialitate absolută.
  • Design rezistent nativ la phishing.
  • Integrare drop-in pentru acces digital și fizic.
Deschide Digi-ID.io Biblioteci
Cu ce se diferențiază de OAuth sau «Conectare cu X»?

Single sign-on-ul tradițional depinde de intermediari de date centralizați. Digi-ID este complet descentralizat: utilizatorul se autentifică semnând o provocare unică direct pe dispozitivul său, eliminând complet serverele terțe din proces.

Este nevoie de un portofel cripto?

Nu, deținerea de cripto este opțională. Multe portofele DigiByte suportă Digi-ID nativ, dar poți folosi și o aplicație de autentificare separată, dedicată exclusiv login-urilor sigure.

Cum o integrez?

Dezvoltatorii pot folosi bibliotecile noastre drop-in pentru Node.js, PHP, Python și Go, sau pot instala pluginul nativ pentru WordPress. Toate resursele se află în hub-ul de dezvoltatori de mai jos.

Hub dezvoltatori

Infrastructură fără compromisuri pentru dezvoltarea UTXO.

Accesează un RPC fiabil, testat în luptă, cu structuri de comisioane fixe și previzibile. DigiByte oferă baza pentru a porni noduri în câteva minute și a construi pe o rețea cu adevărat descentralizată.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

De ce să livrezi pe DGB?
  • Piețe de comisioane previzibile: costuri de tranzacție sub un cent asigură scalare sustenabilă a aplicațiilor.
  • Tranzacții fără aprobări: mecanica nativă UTXO ocolește riscurile de securitate legate de permisiunile ample asupra token-urilor.
  • Execuție fără stare: un mediu UTXO pur permite o dezvoltare simplă, paralelă și componibilă.
Istoric

Peste un deceniu de execuție continuă.

De la activarea timpurie a SegWit până la consensul multi-algoritm. Fiecare upgrade de protocol a fost open source, testat pe mainnet și implementat fără finanțare centralizată.

  1. 2014

    Genesis

    Lansare echitabilă prin minat public; premine de 0,5% distribuit pentru giveaways și dezvoltare.

  2. 2014

    DigiShield

    Reajustarea dificultății în timp real pentru a asigura blocuri stabile.

  3. 2014

    MultiAlgo

    Cinci algoritmi de minat în paralel pentru a preveni centralizarea hashrate-ului.

  4. 2017

    SegWit

    Upgrade de protocol care optimizează capacitatea și remediază maleabilitatea tranzacțiilor.

  5. 2019

    Odocrypt & Dandelion++

    Consens rezistent la ASIC și rutare a tranzacțiilor care păstrează intimitatea.

  6. 2020+

    DigiAssets & Digi-ID

    Integrare la nivel de ecosistem pentru active on-chain și autentificare descentralizată.

  7. 2025

    Core v9.26.4

    Infrastructură de nod modernizată, bazată pe framework-ul Bitcoin Core 26.

    • BIP340 — Schnorr signatures
    • BIP341 — Pay-to-Taproot (P2TR)
    • BIP342 — Tapscript
  8. 2026

    DigiDollar Live on mainnet

    O stablecoin complet non-custodial care generează lichiditate ancorată în USD prin rezerve DGB blocate temporal.

Comunitate

O rețea independentă de constructori globali.

DigiByte este un proiect pur open source, fără o organizație centralizată. Dezvoltarea globală, infrastructura nodurilor și localizarea în peste 40 de limbi sunt menținute integral de voluntari auto-coordonați.

Implică-te: alătură-te ecosistemului auditând cod, localizând documentație, rulând infrastructură sau producând conținut educațional.
