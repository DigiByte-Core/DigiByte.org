Concebida para operação contínua.

A DigiByte é uma blockchain UTXO multi-algoritmo, altamente segura e construída para escalar. Com blocos de 15 segundos e doze anos de operação ininterrupta, a rede oferece uma base testada no terreno para DGB, DigiAssets, autenticação descentralizada Digi-ID e — muito em breve — DigiDollar.

Porquê DigiByte

Concebida para a próxima década do cripto.

Uma cadeia UTXO em prova de trabalho pura que tem entregue pioneirismos desde 2014 — DigiShield, MultiAlgo, SegWit, Odocrypt, Dandelion++, DigiAssets e Digi-ID.

15s
Tempo de bloco

Blocos de 15 s, liquidados depressa.

40× mais rápido que o Bitcoin e 10× mais rápido que o Litecoin. Confirmações em quinze segundos significam que pagamentos POS, depósitos em exchanges e transferências de ativos se finalizam no tempo de ler esta frase — sem o congestionamento ou os picos de taxas das cadeias mais lentas.

5
Consenso

Cinco algoritmos de mineração em paralelo.

A DigiByte protege os blocos com cinco algoritmos de prova de trabalho independentes — SHA-256d, Scrypt, Skein, Qubit e Odocrypt. Cada um visa um ecossistema de mineração diferente (CPU, GPU, FPGA, ASIC) e o Odocrypt re-aleatoriza-se a cada 10 dias para impedir o domínio de ASICs. Atacar a cadeia significa atacar cinco algoritmos ao mesmo tempo.

21B
Oferta

Tecto fixo, escassa por desenho.

A DGB está matematicamente limitada a 21 mil milhões de moedas, emitidas segundo um calendário transparente de redução mensal. Sem taxa para programadores, sem desbloqueios de governança, sem alocações ocultas. Escassez previsível é o que torna um activo monetário credível — e é por isso que cada byte de DGB transporta valor demonstrável e verificável ao longo do tempo.

12 anos no ar
Histórico

Doze anos no ar. Zero falhas.

Desde janeiro de 2014 a mainnet produz blocos a cada quinze segundos sem interrupção. Sem reorgs profundos, sem forks de governança, sem hard forks de emergência. Este uptime, somado a milhares de nós operados de forma independente pelo mundo, é o tipo de histórico operacional de que a infraestrutura crítica precisa — e que quase nenhuma outra cadeia consegue apresentar.

DigiAssets

Emita activos personalizados numa cadeia UTXO segura.

Emita activos digitais personalizados de forma nativa na rede DigiByte, de tokens e stablecoins a registos de identidade. Cada activo é liquidado em 15 segundos e protegido pelo consenso global de prova de trabalho.

Escolha o activo

Token fungível, coleccionável, certificado, voucher — qualquer coisa cunhável numa camada base UTXO.

Defina os metadados

Assine & transmita

Digi-ID

Autenticação de código aberto, sem palavra-passe.

O Digi-ID usa assinaturas criptográficas descentralizadas em vez das palavras-passe tradicionais, protegendo a identidade do utilizador sem intermediários centralizados.

  • Armazenamento local de chaves para privacidade absoluta.
  • Design nativamente resistente a phishing.
  • Integração plug-and-play para acessos digitais e físicos.
Abrir Digi-ID.io Bibliotecas
Qual a diferença em relação ao OAuth ou ao «Entrar com X»?

O single sign-on tradicional depende de intermediários de dados centralizados. O Digi-ID é totalmente descentralizado: os utilizadores autenticam-se assinando um desafio único directamente no dispositivo, removendo por completo servidores de terceiros do processo.

É preciso uma carteira de criptomoedas?

Não, ter criptomoedas é opcional. Muitas carteiras DigiByte já suportam Digi-ID nativamente, mas pode usar uma app autenticadora independente, vocacionada apenas para logins seguros.

Como faço a integração?

Os programadores podem usar as nossas bibliotecas prontas para Node.js, PHP, Python e Go, ou instalar o plugin nativo de WordPress. Os recursos completos estão no hub de programadores abaixo.

Hub de programadores

Infraestrutura sem concessões para desenvolvimento UTXO.

Aceda a uma RPC fiável e testada no terreno, com estrutura de taxas fixa e previsível. A DigiByte fornece a base para subir nós em minutos e construir numa rede genuinamente descentralizada.

~/digibyte — bash
# Get current chain info $ digibyte-cli getblockchaininfo { "chain": "main", "blocks": 21000000, "difficulty": 128456.7, "verificationprogress": 0.99999 } 
// Send 100 DGB via Core RPC (Node.js, no external deps)
import http from 'node:http';
const body = JSON.stringify({ jsonrpc: '1.0', id: 'web', method: 'sendtoaddress',
  params: ['DXXXXXXXXXXXXXXXXXXXXXXXXXXXX', 100] });

const req = http.request({
  hostname: '127.0.0.1', port: 14022, method: 'POST',
  auth: 'rpcuser:rpcpass',
  headers: { 'Content-Type': 'application/json', 'Content-Length': body.length }
}, res => {
  let d = ''; res.on('data', c => d += c);
  res.on('end', () => console.log(JSON.parse(d)));
});
req.write(body); req.end();
# Listen for new blocks (Python, requests)
import requests, time

URL = "http://rpcuser:rpcpass@127.0.0.1:14022"
last = 0
while True:
    info = requests.post(URL, json={
        "jsonrpc": "1.0", "id": "py", "method": "getblockchaininfo", "params": []
    }).json()["result"]
    if info["blocks"] != last:
        print("new block:", info["blocks"])
        last = info["blocks"]
    time.sleep(5)

Porquê lançar na DGB?
  • Mercado de taxas previsível: custos de transacção abaixo de um cêntimo garantem escala sustentável da aplicação.
  • Transacções sem aprovação: a mecânica nativa UTXO contorna os riscos de segurança ligados a permissões amplas de tokens.
  • Execução sem estado: um ambiente UTXO puro permite desenvolvimento simples, paralelo e componível.
Histórico

Mais de uma década de execução contínua.

Da activação precoce do SegWit ao consenso multi-algoritmo. Toda actualização de protocolo foi código aberto, testada em mainnet e implementada sem financiamento centralizado.

  1. 2014

    Génesis

    Lançamento justo via mineração pública; pré-mineração de 0,5 % distribuída em giveaways e desenvolvimento.

  2. 2014

    DigiShield

    Reajuste de dificuldade em tempo real para garantir blocos estáveis.

  3. 2014

    MultiAlgo

    Cinco algoritmos de mineração em paralelo para evitar a centralização do hashrate.

  4. 2017

    SegWit

    Actualização de protocolo que optimiza capacidade e elimina a maleabilidade de transacções.

  5. 2019

    Odocrypt & Dandelion++

    Consenso resistente a ASIC e roteamento de transacções que preserva a privacidade.

  6. 2020+

    DigiAssets & Digi-ID

    Integração em todo o ecossistema para activos on-chain e autenticação descentralizada.

  7. 2025

    Core v9.26.4

    Infraestrutura de nó modernizada, baseada no framework Bitcoin Core 26.

    • BIP340 — Schnorr signatures
    • BIP341 — Pay-to-Taproot (P2TR)
    • BIP342 — Tapscript
  8. 2026

    DigiDollar Live on mainnet

    Uma stablecoin totalmente não custodial que gera liquidez indexada ao dólar via reservas de DGB com bloqueio temporal.

Comunidade

Uma rede independente de programadores globais.

A DigiByte é um projecto puramente de código aberto, sem organização centralizada. Desenvolvimento global, infraestrutura de nós e localização em mais de 40 idiomas são mantidos integralmente por voluntários auto-coordenados.

Participe: entre no ecossistema auditando código, localizando documentação, mantendo infraestrutura ou produzindo conteúdo educativo.
